Queries

debt

Description

Retrieves a debt that you have linked.

Response

Returns a Debt

Arguments
Name Description
id - ID!

Example

Query
query Debt($id: ID!) {
  debt(id: $id) {
    id
    type
    subtype
    providerId
    providerName
    apr
    currency
    currentBalance
    lastStatementBalance
    lastStatementIssueDate
    minimumPayment
    nextPaymentDueDate

}

Mutations

createLinkToken

Description

Returns a link token, which is required as a parameter when initializing Link. Once Link has been initialized, it returns a public token, this should be included in a request to the savePublicToken endpoint so we can access your linked data.

Response

Returns a LinkToken

Arguments
Name Description
type - LinkType!

Example

Query
mutation CreateLinkToken($type: LinkType!) {
  createLinkToken(type: $type) {
    token
    expiryDateTime
  }
}
Variables
{"type": "CREDIT"}
Response
{
  "data": {
    "createLinkToken": {
      "token": "link-sandbox-af1a0311-da53-4636-b754-dd15cc058176",
      "expiryDateTime": "2020-03-27T12:56:34Z"
    }
  }
}

Mutations

savePublicToken

Description

Save a Link public token that has been obtained from the onSuccess callback of Link. The public_token expires after 30 minutes. The public token allows us to access your linked data.

Response

Returns a Boolean

Arguments
Name Description
publicToken - String!

Example

Query
mutation SavePublicToken($publicToken: String!) {
  savePublicToken(publicToken: $publicToken)
}
Variables
{"publicToken": "public-sandbox-5c224a01-8314-4491-a06f-39e193d5cddc"}
Response
{"data": {"savePublicToken": false}}

Types

ID

Description

The ID scalar type represents a unique identifier, often used to refetch an object or as key for a cache. The ID type appears in a JSON response as a String; however, it is not intended to be human-readable. When expected as an input type, any string (such as "4") or integer (such as 4) input value will be accepted as an ID.

Example
"4"

Description

The Int scalar type represents non-fractional signed whole numeric values. Int can represent values between -(2^31) and 2^31 - 1.
